### Project Notes
- **Architecture**: Next.js (frontend) + FastAPI (backend)
- **Backend URL**: `http://localhost:8000`
- **Frontend URL**: `http://localhost:3000`
- **API Key**: The Google Gemini API key is stored locally in `d:\aaaa\foodd\backend\.env`. It has quotes around it, which is parsed correctly by `python-dotenv`.
- **404 error on `/`**: The FastAPI backend is meant for API requests only (`/api/analyze-ingredients`). Visiting `http://localhost:8000` via a browser will return a 404 or a routing message, which is totally normal. Always access the full UI via the frontend at `http://localhost:3000`.
- **Interactive Text Particle**: The main header `Know what you eat.` uses a custom React `<ParticleTextEffect />` component added to `frontend/src/components/ui/interactive-text-particle.tsx`. It relies on HTML5 Canvas and animates the text based on pointer interaction using a custom physical simulation for particle dispersion. The old static particle component was replaced.
- **Dark Mode**: Dark mode was implemented using `next-themes`. A `<ThemeProvider>` wraps the app in `layout.tsx` with `defaultTheme="dark"`. Tailwind `dark:` variants control the UI colors styling in `page.tsx`.
